# Predictive Maintenance MLOps Pipeline

End-to-End MLOps and DevOps Pipeline for Industrial Machine Failure Prediction using AI4I Predictive Maintenance Dataset.

## Features

- DVC Data Versioning
- MLflow Tracking
- FastAPI Deployment
- Docker
- GitHub Actions CI/CD
- Prometheus Monitoring
- Grafana Dashboard
- Evidently AI Drift Detection
- Trivy Security Scanning
